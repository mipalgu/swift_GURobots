NaoV5

public final class NaoV5 : NaoRobot, UpdateableRobot

A read-only nao that reads its values from the whiteboard.

Note that by itself, this struct looks like it doesn’t do anything. This is because, the entirety of its functionality is provided through protocols and protocols extensions. Consider looking through these protocols in order to understand all the functionality that this type provides.

    public init(wb: Whiteboard = Whiteboard(), indexes: NaoWBIndexes = NaoWBIndexes())

    Parameters

    wb

    The whiteboard which this object will fetch its values from. If this parameter is omitted then the global whiteboard is used.
    indexes

    The indexes of the messages within the whiteboard. By default, these indexes are those for the normal global whiteboard. By specifying a value for the indexes, we can allow the use of custom whiteboards. Again, by default, this parameter can be omitted in order to use the global whiteboard indexes.
