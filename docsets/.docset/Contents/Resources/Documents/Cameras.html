  • BottomCameraContainer

    Conforming types have a camera designated the bottom camera.

    This protocol provides convenience functions which make the use of Camera and CameraPivot objects easier. Users of types that conform to this protocol are able to use most of the functionality provided by the Camera and CameraPivot api without having to create their own pivot points and camera objects.

    As the name of the protocol suggests; all the functionality provided by this protocol deals with a specific camera designated the bottom camera.
