SoccerObjectLocations

public struct SoccerObjectLocations : CTypeWrapper

Contains the locations of objects on the soccer field in relation from a source point.

The locations contained within this struct are stored in relation to some other object (usually the robot). As such, these locations are only available if the robot can actually see them. These locations are typically created by querying the messages posted by the Kalman Filter in the whiteboard.
